# Inverted Search in C

A file indexing system developed in C to perform fast keyword-based search across multiple text files using hashing and linked lists.

## Features

* Index multiple text files
* Fast word search
* Display database contents
* Update database
* Save and load indexed data

## Concepts Used

* Hash Tables
* Linked Lists
* File Handling
* Dynamic Memory Allocation

## How to Run

```bash id="0v57ni"
gcc *.c -o search
./search file1.txt file2.txt
```

## Example Use Case

Search a word and quickly find in which files it exists and frequency count.
